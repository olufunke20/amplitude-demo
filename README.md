# amplitude-demo
I implemented event tracking for Landing Page Viewed, Sign Up Clicked, and Request Demo Clicked. I then created a funnel to measure CTA engagement and a behavioral cohort of users who clicked Request Demo. The implementation was connected through my Amplitude workspace created using the provided onboarding link.
 Amplitude Analytics Demo

This project was created as part of a product analytics implementation exercise.
 Features
- Landing page with CTA buttons
- Amplitude event tracking
- Funnel analysis
- Behavioral cohort analysis
